# Git commands

## basics

```sh
git status
git log

git add <your files to add>
git commit -m "your commit message"

git push
git pull
```

## configuration

```sh
git config --list
```

```sh
git config user.name "your name"
```

```sh
git config user.email "your_email@example.com"
```
